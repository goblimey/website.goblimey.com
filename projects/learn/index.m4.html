The UNIX Learn System

Learn is a Computer-Based Teaching tool that gives basic training in the UNIX command line interface. Apple's MacOS and IOS operating systems are based on UNIX, as is Linux and Android.

The learn software can be run in various environments including Microsoft's Windows Home.

UNIX-based systems are widely used on cloud computers, accessed via the command-line interface that the learn software teaches.
