Courses

UNIX and Linux commands self-teaching software

Learn is Computer-Based Teaching software that gives basic training in using commands on UNIX and Linux. (Linux is a version of UNIX.) The learn software is free and you can run it on a Windows PC, so you can learn to use a UNIX system without having access to one.

You will find the instructions for setting up and running the learn software here.

Once you have completed the learn courses, you can hire a Linux system in the cloud for as little as $5 per month as described here.
